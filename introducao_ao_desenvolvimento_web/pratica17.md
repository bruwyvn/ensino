# Prática 17

Utilização do `while`

```
<?php

$counter = 0;

while ($counter <= 7) {
  echo "<span>" . $counter . "</span><br>";
  $counter++; // é a mesma coisa de fazer $counter = $counter + 1;
}
```


Utilização do `for`
```
<?php

for ($i = 0; $i < 10; $i++) {
  echo "<span>" . $i . "</span><br>";
}
```
