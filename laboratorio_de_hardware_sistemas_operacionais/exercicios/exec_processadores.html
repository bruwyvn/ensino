b) Memória externa do computador

d) nenhuma das alternativas

2) Sobre os processadores, marque a alternativa errada

a) O ciclo é repetido até o programa terminar
b) um processador x86 pode executar programas ARM, mas um processador ARM não consegue executar programas x86.
c) Cada CPU tem um conjunto específico de instruções que ela consegue executar
d) Processadores não possuem memórias próprias, usam sempre a memória DRAM

3) Sobre os processadores, marque a alternativa correta

a) Como o tempo para acessar a memória para buscar uma instrução ou palavra dos operandos não é muito maior do que o tempo para executar uma instrução, todas as CPUs têm alguns registradores internos para armazenamento de variáveis e resultados temporários.
b) O conjunto de instruções geralmente contém instruções para carregar uma palavra da memória para um registrador e apenas isso.
c) Além dos registradores gerais usados para armazenar variáveis e resultados temporários, a maioria dos computadores tem vários registradores especiais que são visíveis para o programador.
d) Nenhum registrador do processador é visível pelo programador

4) O que é o contador de programa?

a) contém o endereço de memória da próxima instrução a ser buscada. Após essa instrução ter sido buscada, o contador de programa é atualizado para apontar a próxima instrução
b) PSW (Program Status Word — palavra de estado do programa). Esse registrador contém os bits do código de condições, que são estabelecidos por instruções de comparação, a prioridade da CPU, o modo de execução (usuário ou núcleo) e vários outros bits de controle.
c) aponta para o topo da pilha atual na memória. A pilha contém uma estrutura para cada rotina que foi chamada, mas ainda não encerrada.
d) um conjunto específico de instruções que ela consegue executar

4) O que é o topo de pilha?

a) contém o endereço de memória da próxima instrução a ser buscada. Após essa instrução ter sido buscada, o contador de programa é atualizado para apontar a próxima instrução
b) PSW (Program Status Word — palavra de estado do programa). Esse registrador contém os bits do código de condições, que são estabelecidos por instruções de comparação, a prioridade da CPU, o modo de execução (usuário ou núcleo) e vários outros bits de controle.
c) aponta para o topo da pilha atual na memória. A pilha contém uma estrutura para cada rotina que foi chamada, mas ainda não encerrada.
d) um conjunto específico de instruções que ela consegue executar

5) Assine a alternativa errada no que tange ao conceito de pipeline

a) Para melhorar o desempenho, os projetistas de CPU há muito tempo abandonaram o modelo simples de buscar, decodificar e executar uma instrução de cada vez.
b) Muitas CPUs modernas têm recursos para executar mais de uma instrução ao mesmo tempo.
c) Por exemplo, uma CPU pode ter unidades de busca, decodificação e execução separadas, assim enquanto ela está executando a instrução n, poderia também estar decodificando a instrução n + 1 e buscando a instrução n + 2.
d) nenhuma das alternativas

6) Sobre o modo núcleo, marque a correta

a) A maioria das cpus possuem três modos, núclo, usuário e kernel
b) PSW não tem nada a ver com o modo núcleo
c) Em computadores de mesa e servidores, o sistema operacional normalmente opera em modo usuário, dando a ele acesso a todo o hardware.
d) Quando operando em modo núcleo, a CPU pode executar todas as instruções em seu conjunto de instruções e usar todos os recursos do hardware.

7) Sobre o modo usuário, marque a correta

a) A maioria das cpus possuem três modos, núclo, usuário e kernel
b) PSW não tem nada a ver com o modo núcleo
c) Em computadores de mesa e servidores, o sistema operacional normalmente opera em modo usuário, dando a ele acesso a todo o hardware.
d) Geralmente, todas as instruções envolvendo E/S e proteção de memória são inacessíveis no modo usuário. Alterar o bit de modo PSW para modo núcleo também é proibido, claro.
